# Mobile Money Simulation (C)

A command-line mobile money application written in C that simulates common mobile money operations.

## Features
- User authentication with PIN validation
- Send and transfer money across networks
- Buy airtime
- Menu-driven interface
- Transaction logging using file I/O

## Technologies Used
- C
- Standard I/O (stdio.h)
- String handling (string.h)
- File handling

## What I Learned
- Writing modular programs using functions
- Using conditional logic and switch statements
- Handling user input safely
- Logging data to files
- Simulating real-world systems using C# c-mobile-money-app
